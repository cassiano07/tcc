CRIAR FORMULARIO PARA ENVIAR ARQUIVO

Nesse campo o usuário fara o envio de seu arquivo para que seja feita a análise de seus dados para confirmarmos se esses dados estão aptos para serem processados. OBS.: Esses arquivos devem ser separados por vírgulas, porém, em cada informação dos dados não podem conter vírgulas. Segue o exemplo abaixo, como cor AZUL o que PODE conter vírgula e cor VERMELHA o que NÃO PODE conter vírgula. Ex.:

Nessa etapa (se os dados estiverem aptos) faremos a separação de todas as colunas e as classificaremos por números e textos, nas quais o usuário deverá escolher uma forma de texto e outra de número e qual forma deverá ser feito. Exemplo: Soma, multiplicação, contagem ou nenhum.

Se tudo ocorreu como deveria, o sistema encaminhara o usuário para uma próxima página onde o usuário encontrara o seu arquivo em forma de dashboard (painel de gráficos), podendo fazer download para o seu aparelho ou criar uma conta e deixar salvo em nosso site.
