Nesse campo o usuário fara o envio de seu arquivo para que seja feita a análise de seus dados para confirmarmos se esses dados estão aptos para serem processados.

As informações contidas nesses arquivos devem ser separadas por vírgulas, porém, no meio dessas informações não podem conter vírgulas. O exemplo abaixo mostra a forma correta de como as informações devem estar dentro do seu arquivo, o primeiro exemplo mostra a forma correta onde as virgulas devem ficar, que no caso é delimitando as informações de cada coluna e no segundo exemplo a cor VERMELHA sinaliza onde NÃO PODE conter vírgulas, pois o nosso sistema entendera que você quer dividir aquela informação em duas colunas.

Nessa etapa (se os dados estiverem aptos) faremos a separação de todas as colunas e as classificaremos por números e textos, nas quais o usuário deverá escolher uma forma de texto e outra de número e qual forma deverá ser feito. Exemplo: Soma, multiplicação, contagem ou nenhum. Se tudo ocorreu como deveria, o sistema encaminhara o usuário para uma próxima página onde o usuário encontrara o seu arquivo em forma de dashboard (painel de gráficos), podendo fazer download para o seu aparelho ou criar uma conta e deixar salvo em nosso site.

Abaixo o usuário poderá ver os gráficos que estarão disponivel em nossa plataforma e serão apresentados após todo o processo de análise.
